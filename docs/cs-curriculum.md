# CS + Algorithm Interactive Curriculum
> 80/20 원칙. Claude Q&A 기반. 출퇴근길 활용.
> 대상: 임베디드 비전 개발자 + 기초 개발자
> 기간: 7.5주 (4/7 ~ 5/29)

## 학습 방식
- **출근길 (~1.5h)**: CS 핵심 개념 Q&A
- **퇴근길 (~1.5h)**: 알고리즘 개념/풀이 전략 Q&A
- **집**: 알고리즘 실제 코딩 (짧게)
- **방법**: Claude에게 "이번 주 CS 주제 시작해줘" → 질의응답 → 모르는 부분 즉시 파악 → 핵심만 확실히

## 진행 규칙

1. **매일 출근길**: "이번 주 CS 주제 Q&A 시작" → Claude가 핵심 개념 질문 → 모르면 설명 → 알면 다음
2. **매일 퇴근길**: "이번 주 알고리즘 주제 Q&A" → 개념 확인 + 문제 풀이 전략 토론
3. **집에서 20-30분**: 추천 문제 1개 코딩. 15분 안에 안 풀리면 풀이 보고 이해.
4. **주말**: 그 주 체크리스트 점검. 빈 칸 있으면 보충.
5. **체크 기준**: Claude와 Q&A에서 막힘 없이 설명할 수 있으면 체크.
